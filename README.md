# Game_Jam

Welcome to the world of Multiplayer shooter brawl, an exciting multiplayer game where players can take on the role of either the Hero or the Ghost. The Hero's mission is to eliminate all the robots scattered across the game world, while the Ghost's objective is to control the robots and eliminate the Hero. This intense battle between good and evil will test your skills and strategy.
